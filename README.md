## 1️⃣ Getting started

To get the server running locally:

- Clone this repo
- **npm install** to install all required dependencies
- **npm server** to start the local server
- **npm test** to start server using testing environment

### Framework

- NodeJS
- Express
- Knex
- Jest
- Supertest
- Jsonwebtoken
- Bcryptjs
